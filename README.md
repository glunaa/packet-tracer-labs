# 📡 Packet Tracer Labs — Giovanni Luna

A collection of hands-on Cisco Packet Tracer labs covering core networking concepts for **CompTIA Network+** and **CCNA** exam prep.

Each lab includes objectives, a topology diagram, and step-by-step IOS configuration commands.

---

## 🗂️ Labs

| # | Topic | Key Concepts |
|---|-------|-------------|
| 01 | [Basic LAN + Switch Config](./01-basic-lan/) | Hostnames, passwords, VTY, SSH, port security |
| 02 | [VLANs & Trunking](./02-vlans-trunking/) | VLAN creation, access ports, 802.1Q trunk |
| 03 | [Inter-VLAN Routing](./03-inter-vlan-routing/) | Router-on-a-stick, subinterfaces, encapsulation |
| 04 | [DHCP + DNS](./04-dhcp-dns/) | DHCP pools, exclusions, DNS server config |
| 05 | [Static Routing & Default Routes](./05-static-routing/) | ip route, default gateway, floating static |
| 06 | [OSPF](./06-ospf/) | Single-area OSPFv2, DR/BDR, passive interfaces |

---

## 🛠️ Tools

- **Cisco Packet Tracer** 8.x (free via [Cisco Skills For All](https://skillsforall.com/))
- IOS commands tested on Cisco 2960 switches and 2911 routers

## 📌 How to Use

1. Open the `.pkt` file in Packet Tracer
2. Read the lab `README.md` for objectives and topology
3. Configure devices using the step-by-step commands
4. Use `show` commands to verify your work
